# 🎓 Portofolio Hasil Kerja Praktek - ICT Universitas Teuku Umar

Repositori ini berisi kumpulan poyek website yang kami kembangkan selama menjalani masa **Kerja Praktek (KP)** di ICT Universitas Teuku Umar.

Tujuan dari repositori ini adalah sebagai bentuk pertanggungjawaban laporan KP serta sebagai portofolio pribadi untuk memperlihatkan kemampuan *problem solving* dan keahlian kami sebagai *developer*.

#### 📝 Deskripsi Proyek
Proyek ini bertujuan untuk memngimplementasikan desain (UI/UX) dari Figma ke HTML CSS sekaligus menghubungkannya ke database menggunakan PHP.

#### 🛠️ Teknologi yang Digunakan
* **Figma:** Untuk desain halaman. 
* **HTML5:** Untuk menyusun struktur dasar form.
* **CSS3:** Untuk styling, penataan layout menggunakan Flexbox, dan pembuatan antarmuka yang responsif.
* **PHP:** Untuk struktur file *backend* sederhana.

## 🔧 Cara Menjalankan Proyek di Komputer Lokal

Jika ingin mencoba menjalankan kode dari repositori ini di komputermu sendiri:

1.  **Clone Repositori:**
    ```bash
    git clone [https://github.com/username-github-kamu/nama-repositori.git](https://github.com/username-github-kamu/nama-repositori.git)
    ```
2.  **Masuk ke Folder Proyek:**
    ```bash
    cd nama-folder-proyek
    ```
3.  **Jalankan:**
    Karena ada file PHP, pastikan untuk memindahkan folder proyek ke dalam direktori server lokal seperti `htdocs` (jika memakai XAMPP) dan akses melalui browser via `localhost/nama-folder-proyek`.
